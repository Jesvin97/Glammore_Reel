## WEEK 1 CHECKLIST

**Reel 1 (Tue): "Keratin ruined your hair? No. Bad execution did."**
**Reel 8 (Fri): "Salon myth: expensive treatment means better result."**

### Pre-Shoot (Monday)
- [ ] Script finalized for both reels
- [ ] Client confirmed (Reel 1)
- [ ] Client confirmed (Reel 8)
- [ ] Location scouted (good lighting, quiet for voiceover)
- [ ] Equipment tested (phone, tripod, ring light, external mic)
- [ ] Props ready (keratin product, before/after photos)

### Shooting Day (Tuesday)
- [ ] Reel 1 filmed (before footage, process, after, client reaction) — 2–3 hours
- [ ] Reel 8 filmed (comparison footage, testimonials) — 2–3 hours
- [ ] Voiceover recorded for both reels (separate session, quiet room)
- [ ] Files backed up to cloud storage

### Editing (Wed–Thu)
- [ ] Reel 1 edited (timelapse cuts, voiceover synced, text overlays)
- [ ] Reel 1 color graded (brightness, saturation matched)
- [ ] Reel 1 final review (pacing, messaging, audio quality)
- [ ] Reel 8 edited (comparison footage, transitions smooth)
- [ ] Reel 8 color graded
- [ ] Reel 8 final review

### Posting (Thursday)
- [ ] Reel 1 caption written (hook + body + CTA + link)
- [ ] Reel 1 thumbnail selected (most compelling frame)
- [ ] Reel 1 scheduled for Tuesday 6pm IST
- [ ] Reel 8 caption written
- [ ] Reel 8 thumbnail selected
- [ ] Reel 8 scheduled for Friday 6pm IST

### Response (Tue + Fri after posting)
- [ ] Reel 1: Monitor first hour, respond to all comments
- [ ] Reel 1: Track reach at 24h, 48h, 72h
- [ ] Reel 8: Monitor first hour, respond to all comments
- [ ] Reel 8: Track reach at 24h, 48h, 72h

### Weekly Sync (Friday evening)
- [ ] Review both reels' performance (reach, engagement, saves, shares)
- [ ] Check competitor activity (Naturals, LA PIERA, Zayana)
- [ ] Count DMs asking about booking
- [ ] List feedback/comments (what resonated?)
- [ ] Plan adjustments for Week 2 (if needed)

**WEEK 1 GOAL:** Post 2 reels on schedule, monitor reach, respond to comments

**Week 1 Metrics to Track:**
